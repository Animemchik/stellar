# Keyword

Keywords can only be used in their correct contexts. They cannot be used as the names of:

- [Items](items)
- Variables and function arguments
- Fields and variants
- Associated functions

> **<sup>Lexer:<sup>**\
> KW_AS             : `as`\
> KW_BREAK          : `break`\
> KW_CONTINUE       : `continue`\
> KW_ELSE           : `else`\
> KW_ENUM           : `enum`\
> KW_FALSE          : `false`\
> KW_FOR            : `for`\
> KW_FN             : `fun`\
> KW_IF             : `if`\
> KW_IMPL           : `impl`\
> KW_LOOP           : `loop`\
> KW_MUT            : `mut`\
> KW_PUB            : `pub`\
> KW_RETURN         : `return`\
> KW_STRUCT         : `struct`\
> KW_TRAIT          : `trait`\
> KW_TRUE           : `true`\
> KW_TYPE           : `type`\
> KW_WHERE          : `where`\
> KW_WHILE          : `while`\
> KW_SWITCH         : `switch`\
> KW_DEFER          : `defer`
